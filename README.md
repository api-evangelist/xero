# XeroThis is a repo for managing machine readable artifacts for the [Xero APIs](http://www.xero.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Xero APIs](http://www.xero.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.xero.com/)- [Developer](http://www.xero.com/)- [Twitter](https://twitter.com/xero)- [GitHub](https://github.com/XeroAPI)The [APIs.json](https://github.com/api-evangelist/xero/blob/master/apis.json) index provides links to all the relevant links for the Xero APIs, including the OpenAPI and Postman if they are present.
