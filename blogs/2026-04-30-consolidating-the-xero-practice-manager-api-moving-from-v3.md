For several years, Xero has maintained two versions of the Xero Practice Manager (XPM) API: v3.0 (Integer-based) and v3.1 (UUID-based). To ensure we can continue to scale our infrastructure and provide better performance, we are now beginning the process of consolidating onto a single version. Migrate to v3.1 by April 30, 2027, to unlock greater integration efficiency and features While v3.1 has been available for some time, this 12-month notice marks the start of the formal deprecation period for v3.0.
